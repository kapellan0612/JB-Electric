# JB Electric — Website UI Kit

## Overview
High-fidelity recreation of the JB Electric marketing website. Built with React + Babel, modular JSX components.

## Screens
- **Homepage** — Hero, services grid, trust bar, contact CTA
- **Services** — Service detail cards
- **Contact** — Contact form + info panel

## Components
- `Header.jsx` — Nav with logo, phone, CTA button
- `Hero.jsx` — Full-bleed navy hero with bolt divider motif
- `Services.jsx` — 4-up service card grid with descriptions
- `Footer.jsx` — Contact info, trust markers, copyright

## Usage
Open `index.html` in a browser. Click nav links to switch screens.
